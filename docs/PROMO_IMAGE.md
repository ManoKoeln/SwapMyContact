# Promo Image für IAP (Apple Guideline 2.3.2)

Dieses Promo‑Bild ist eindeutig und kein App‑Icon. Es bewirbt Premium (unbegrenzte Profile) und kann als Promotional Image im IAP verwendet werden.

- Quelle (SVG): `assets/promo-premium.svg`
- Vorschau/Export: `assets/promo-preview.html`
- Ziel: PNG 1024×1024 (App Store Connect → In‑App Purchase → Promotional Image)

## Export‑Hinweise (macOS)
- SVG in Safari öffnen → Ablage → Exportieren → Format: PNG → Größe 1024×1024
- Alternativ in Vorschau öffnen und als PNG exportieren.

## Upload in App Store Connect
1. App Store Connect → Meine Apps → In‑App Purchases → Wähle Premium‑Produkt
2. Promotional Image: PNG (1024×1024) hochladen
3. Speichern

Falls keine Promotion geplant ist, kannst du das Promotional Image im Produkt löschen, um die Richtlinie zu erfüllen.
